# Recruitment Platform Funnel & Cohort Analysis

## Overview
This project demonstrates advanced Data Analyst skills using Python and SQL. It simulates a recruitment platform's user lifecycle to generate business insights regarding:
- **Funnel Conversion**: Identifying drop-off points in the hiring process.
- **User Retention**: Analyzing monthly cohort retention using a heatmap.

## Files
- `recruitment_analysis.py`: Main executable script containing Mock Data Generation, SQL Analysis, and Cohort Analysis.
- `cohort_analysis.png`: Visualization output generated by the script.

## key Technologies
- **Python**: Pandas, NumPy, Matplotlib, Seaborn, Faker
- **SQL**: SQLite (Window Functions, CTEs)

## How to Run
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the analysis:
   ```bash
   python recruitment_analysis.py
   ```
3. View the generated Report in the console and the Heatmap in `cohort_analysis.png`.
